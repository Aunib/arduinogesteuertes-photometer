# arduinogesteuertes-photometer
Ein arduinogesteuertes Photometer für den Gebrauch an Schulen, entwickelt im Rahmen der Vorbereitung auf meine MSA-Präsentationsprüfung im Schuljahr 2019/20. Die Idee stammt von der Technologiestiftung Berlin.
